erlang安装教程

发表于 2017-01-23 | 分类于 erlang | 阅读次数

操作系统： centos

如果需要使用debugger工具先进行此步骤

%%if use debugger you need install wxwidgets
%%wxWidgets3.0.2.tar.gz
./configure --with-opengl --enable-debug --enable-unicode
make && make install

vim ~/.bash_profile
%%bash_profile文件中添加以下内容
LD_LIBRARY_PATH=/usr/local/lib
export=LD_LIBRARY_PATH
保存

%% 初始化该文件,使它生效
source ~/.bash_profile

%% 验证初始化是否生效  打印出路径就说明成功
echo $LD_LIBRARY_PATH

安装erlang步骤

%% install erlang
%% 安装 erlang
%% otp-R17.4.tar.gz  wget http://www.erlang.org/download/otp_src_17.3.tar.gz
tar zxvf otp_src_17.3.tar.gz
cd otp_src_17.3
./configure  --with-ssl --enable-threads --enable-smp-support --enable-kernel-poll --enable-hipe
make && make install
到此步骤没有报错的话 应该是安装成功

可能遇到的错误

configure: error: No curses library functions found
原因： 缺少ncurses安装包
解决方法 yum -y install ncurses-devel.x86_64
－－－－－－－－
Could not run GTK+ test program, checking why... ->
原因：缺少GTK安装包
解决方法：
yum install wxGTK-devel
yum install gtk+-devel.x86_64
yum install gtk2-devel.x86_64
－－－－－－－－
error: openGL ...
原因:缺libGLU安装包
解决方法：yum install mesa-libGLU-devel.x86_64
－－－－－－－－
c++ gcc error
原因：缺gcc安装包
解决方法：yum install gcc-c++
－－－－－－－－
crypto         : No usable OpenSSL found
原因：openssl
解决方法：yum install openssl-devel.x86_64
－－－－－－－－
odbc library - link check faild
原因：缺少unixODBC安装包
解决方法：yum install unixODBC-devel
